Setup guide — total time: ~90 minutes
1
Set up Stripe ~20 min
stripe.com
  1. Go to stripe.com and create an account with builtbyaaronf@gmail.com.
  2. Complete business verification (you'll need your name, address, and bank account details for payouts).
  3. Go to Invoicing → Invoice templates and set up your FORGE branding (logo, colours, payment terms: "Due on receipt").
  4. Create three Products: Quick Deploy — £64.99, Standard Deploy — £444.99, Full Deploy — £949.99. These become line items on invoices.
  5. Note your Stripe API key (Developers → API Keys → Restricted key with Invoice write access) — you'll need it for Zapier.
Fee: Stripe charges 1.5% + 25p per transaction (UK cards) or 2.9% + 25p (international). On a £444.99 Standard Deploy that's ~£6.95 — worth it for zero manual work.
2
Build the Tally form ~20 min
tally.so
  1. Sign up at tally.so (free plan is sufficient).
  2. Create a new form titled "FORGE — Get Started".
  3. Add these fields in order:
    • Business name (Short text, required)
    • Your email (Email, required)
    • Business location / area (Short text, required)
    • Which package? (Dropdown: Quick Deploy £64.99 / Standard Deploy £444.99 / Full Deploy £949.99, required)
    • Anything else? (Long text, optional)
  4. Set up a Thank you page message: "We've received your brief. Your invoice will arrive within 2 hours. Work begins the moment payment clears."
  5. Copy the form's share URL — you'll link to it from the FORGE site, replacing the Calendly links on the package cards.
  6. To pre-select the package, Tally supports URL parameters: ?package=Quick+Deploy — use this when you link each package CTA to the form so the right package is pre-filled.
Tip: Keep Calendly for the nav and hero CTA (discovery call path). Use the Tally form for the package card CTAs (buy now path). Two conversion paths: high-intent buyers go straight to form; uncertain visitors book a call.
3
Wire it up with Zapier ~30 min
zapier.com
  1. Sign up at zapier.com (Starter plan at ~$20/mo covers this).
  2. Create Zap 1 — Stripe Invoice:
    Trigger: Tally → New Submission
    Action: Stripe → Create Invoice — map Customer Email from Tally, look up the product by package name, set auto-advance to true so it sends immediately.
  3. Create Zap 2 — HubSpot Contact + Deal:
    Trigger: Tally → New Submission (same trigger, second Zap)
    Action 1: HubSpot → Create/Update Contact — map email, name (from business name), company.
    Action 2: HubSpot → Create Deal — deal name = business name + package, amount = package price, stage = "Invoice Sent".
  4. Create Zap 3 — Your notification:
    Trigger: Tally → New Submission
    Action: Gmail → Send email to builtbyaaronf@gmail.com with subject "New FORGE order — [Business Name] — [Package]" and all form fields in the body. You know instantly when a brief comes in.
  5. Test all three Zaps with a real test submission before going live.
Cost: Zapier Starter ~$20/mo. At £64.99 per Quick Deploy, one sale per month more than covers it. At £444.99 per Standard Deploy, the ROI is instant.
4
Update the FORGE site CTAs ~15 min
index.html

  3. Update the #contact section to show both paths: "Book a call" (Calendly) and "Or fill in your brief and we'll send your invoice within 2 hours" (Tally).
