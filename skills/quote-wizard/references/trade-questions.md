# Trade-Specific Follow-Up Questions

For Step 2 of the quote wizard. Use 1–2 questions max per business type. Pick the most relevant based on selected services. Keep labels short — these appear inline in a multi-step form.

---

## plumber
- **"Is this a repair or a new installation?"** — select: Repair / New installation / Not sure
- **"What area is affected?"** — select: Bathroom / Kitchen / Boiler/heating / Outside / Multiple areas
- **"Is this urgent? (e.g. active leak or no hot water)"** — select: Yes, urgent / No, can wait

## electrician
- **"Is this a fault or a new installation?"** — select: Fault / New installation / Inspection/certificate needed
- **"How many rooms or circuits are affected?"** — select: Just one / 2–5 / Whole property
- **"Is your power currently out?"** — select: Yes / No / Partial

## builder
- **"What type of work is this?"** — select: Extension / Conversion / New build / Structural repair / Renovation
- **"Is planning permission already in place?"** — select: Yes / No / Not required / Not sure

## landscaper
- **"What's the approximate size of the area?"** — select: Small (under 30m²) / Medium (30–100m²) / Large (100m²+)
- **"Is there existing hard landscaping to remove or work around?"** — select: Yes / No / Not sure

## cleaner
- **"What type of property is this?"** — select: House / Flat / Office / Commercial / Other
- **"Is this a one-off or regular clean?"** — select: One-off / Weekly / Fortnightly / Monthly

## painter
- **"Is this interior, exterior, or both?"** — select: Interior / Exterior / Both
- **"How many rooms or surfaces need painting?"** — text input (short): placeholder "e.g. 3 bedrooms + hallway"

## roofer
- **"Is there an active leak or structural damage?"** — select: Yes, urgent / No, preventative / Not sure
- **"Approximately how old is the roof?"** — select: Under 10 years / 10–25 years / 25+ years / Unknown

## carpenter
- **"What type of woodwork is needed?"** — select: Fitted furniture / Doors/frames / Flooring / Decking / Bespoke item / Repair
- **"Do you have materials or should we supply?"** — select: Please supply / I have materials / Mix of both

## tiler
- **"What surface needs tiling?"** — select: Bathroom / Kitchen / Floor / Wall / Outdoor / Multiple
- **"Approximate area in m²?"** — text input (short): placeholder "e.g. 12m²"

## handyman
- **"How many tasks need doing?"** — select: Just one / 2–3 things / Several jobs
- **"Is there any urgency?"** — select: Yes, ASAP / Flexible / Booking ahead

## beauty
- **"Is this your first appointment with us?"** — select: First time / Returning client
- **"Do you have any known allergies or sensitivities we should know about?"** — text input (short): placeholder "e.g. latex, certain dyes (optional)"

## general
- **"What is your timeline for this work?"** — select: ASAP / Within a month / Just exploring / Flexible
- **"Is there anything else we should know before quoting?"** — text input (short): placeholder "e.g. access issues, existing conditions"
